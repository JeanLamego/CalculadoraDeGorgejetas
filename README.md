## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github

## 💻 Projeto

A calculadora de gorjetas foi feita baseada em um desafio do FrontEndMentor com o intuito de praticar minhas habilidades em HTML, CSS e principalmente Javascript. Foi um projeto desafiador pois eu não tinha muito conhecimento da DOM e foi super legal conseguir ir superando os desafios pouco a pouco com ajuda das comunidades de desenvolvedores e muito estudo.
